## Quick Start

clone this repo

```
git clone https://gitee.com/roft/helper-starter-kit.git
cd helper-starter-kit
yarn
yarn test
```

or using app-starter-kit

```
yarn add @roft/app-starter-kit global
kit init my-cool-app helper
cd my-cool-app
yarn test
```

Finally, View on the console.
