## 快速开始

拷贝这个仓库

```
git clone https://gitee.com/roft/present.git
cd present
yarn
yarn test
```

或者使用 app-starter-kit

```
yarn add @roft/app-starter-kit global
kit init my-cool-app helper
cd my-cool-app
yarn test
```

最后, 在控制台预览.
